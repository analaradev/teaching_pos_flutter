## Contexto

Repositorio orientado a ensenanza y practica guiada de desarrollo movil mediante una aplicacion tipo POS.

## Proposito

- Mostrar estructura base de una aplicacion POS.
- Trabajar pantallas, navegacion y formularios.
- Explicar logica basica de operaciones de venta.
- Servir como recurso para sesiones de aprendizaje.
